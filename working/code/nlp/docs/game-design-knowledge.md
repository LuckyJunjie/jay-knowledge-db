# 游戏策划知识库 (Game Design Knowledge)

## 1. 游戏机制 (Game Mechanics)

### 核心循环
- **资源获取** → **资源转换** → **资源消耗**
- **挑战** → **学习** → **精通**

### 反馈系统
- 即时反馈：碰撞、得分、动画
- 延迟反馈：升级、解锁、成就
- 社交反馈：排行榜、社区

### 难度曲线
```
难度 = 玩家技能 / 挑战需求
保持 ≈ 0.7-0.9 范围
```

## 2. 关卡设计 (Level Design)

### Fogg 教授行为模型
B = MAP (Motivation x Ability x Prompt)
- 高动机 + 高能力 + 触发 = 行为
- 教学关卡：降低 Ability 门槛
- 挑战关卡：提升动机

### 学习流程
1. **引导** - 展示机制
2. **练习** - 无压力尝试  
3. **应用** - 组合机制
4. **精通** - 创意表达

## 3. 玩家心理学

### 心流理论 (Flow)
- 技能过高 → 无聊
- 技能过低 → 焦虑
- 进入心流 → 适度挑战

### 内在动机
- 自主感：选择自由
- 胜任感：成长反馈
- 归属感：社区连接

### 上瘾机制（谨慎使用）
- 可变奖励 > 固定奖励
- 社交比较
- 损失厌恶 (FOMO)
- 进度投资

## 5. 经典设计模式

- **MDA Framework**: Mechanics → Dynamics → Aesthetics
- **Core Loop**: 核心 ↔ 外围 → 社交
- **Hook Model**: Trigger → Action → Reward → Investment
- **Bartle Taxonomy**: Achievers, Explorers, Socializers, Killers
