# 技能分类系统 (Skill Classification)

## 四大技能类别

### 1. 技术技能 (Technical Skills)

| 技能 | 描述 | 级别 |
|---|---|---|
| 编程 | Python, JavaScript, Go | 基础→高级 |
| 数据库 | SQL, Redis, PostgreSQL | 基础→专家 |
| 云服务 | AWS, GCP, Azure | 基础→专家 |
| 机器学习 | 传统ML, 深度学习 | 基础→专家 |
| 前端开发 | React, Vue, Godot | 基础→专家 |
| 后端开发 | API, 微服务 | 基础→专家 |
| DevOps | Docker, K8s, CI/CD | 基础→专家 |

### 2. 业务技能 (Business Skills)

| 技能 | 描述 | 级别 |
|---|---|---|
| 产品设计 | 需求分析, 原型 | 基础→专家 |
| 项目管理 | Agile, Scrum | 基础→专家 |
| 数据分析 | SQL, Excel, BI | 基础→专家 |
| 用户研究 | 访谈, 问卷 | 基础→高级 |
| 增长运营 | 漏斗, A/B | 基础→高级 |

### 3. 通用技能 (Soft Skills)

| 技能 | 描述 | 级别 |
|---|---|---|
| 沟通 | 写作, 演讲 | 基础→专家 |
| 问题解决 | 结构化思维 | 基础→专家 |
| 学习能力 | 元认知 | 基础→专家 |
| 时间管理 | GTD, 番茄 | 基础→专家 |

### 4. AI/机器学习技能 (AI/ML Skills)

| 技能 | 描述 | 级别 |
|---|---|---|
| LLM应用 | Prompt工程, RAG | 基础→专家 |
| 模型微调 | LoRA, RLHF | 基础→高级 |
| 向量检索 | Embedding, ANN | 基础→高级 |
| Agent设计 | ReAct, ToolUse | 基础→专家 |
| 多模态 | VLM, TTS | 基础→高级 |

## 技能评估矩阵

```
熟练度: L1(了解) → L2(会用) → L3(熟练) → L4(专家) → L5(创造)

能力 = 技术深度 × 业务广度 × 通用软度
```

     ↓              ↓
  移动端        WebGL
```

### 后端技能树
```

  ↓    ↓      ↓      ↓

   ↓      ↓       ↓       ↓
