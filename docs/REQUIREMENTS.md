# Jay Knowledge DB - Requirements

## Project Background
Automated knowledge management system integrating finance, gaming, AI/Agent, robotics.

## Functional Requirements

### FR-001: Knowledge Collection
- FR-001.1: Financial macro data (power, exports, PMI)
- FR-001.2: Financial news real-time collection
- FR-001.3: ArXiv paper acquisition
- FR-001.4: Game dev blog/GitHub trending
- FR-001.5: Industry conference info (GDC, ICRA)

### FR-002: NLP Pipeline
- FR-002.1: Multi-domain classification
- FR-002.2: Entity recognition
- FR-002.3: Financial sentiment analysis
- FR-002.4: Auto summary (50 chars)
- FR-002.5: Text embedding
- FR-002.6: Knowledge graph relations

### FR-003: Statistics Engine
- FR-003.1: Time-series storage & MA
- FR-003.2: Expectation comparison
- FR-003.3: Macro sentiment index (-100 to +100)
- FR-003.4: Chart config generation

### FR-004: Human Portal
- FR-004.1: Sentiment dashboard
- FR-004.2: Key indicator charts
- FR-004.3: News feed
- FR-004.4: Knowledge detail page

### FR-005: Agent API
- FR-005.1: /api/v1/search endpoint
- FR-005.2: /api/v1/context endpoint
- FR-005.3: Citation tracking
- FR-005.4: Stats snapshot
